\# 📋 Descripción



Districarnes es un sistema de comercio electrónico desarrollado en Java utilizando Programación Orientada a Objetos.



El sistema permite:



\- Gestión de productos

\- Gestión de usuarios

\- Inicio de sesión

\- Carrito de compras

\- Gestión de pedidos

\- Métodos de pago

\- Interfaz gráfica funcional con Swing



El proyecto fue desarrollado aplicando los pilares fundamentales de la Programación Orientada a Objetos.
